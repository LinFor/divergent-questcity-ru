Вывод интерфейса OnRiv внутри страниц сайта

----------------------------------------------------

Может осуществляеться по средствам PHP функции "include".
Если поместить в код страницы, ниже приведённую конструкцию, то на ней будет отображена категория имеющая ID "category_04_02_2016_12_31_37_24". То же самое можно сделать и с отдельным объектом, указав его ID в переменной "$obj".
На одной странице можно вывести только одну данную конструкцию. ID объекта и категории можно увидеть в редакторе панели управления, после опции "Активировать". 

----------------------------------------------------

$cat = 'category_04_02_2016_12_31_37_24';         
// ID категории, если требуется вывести одну категорию
$obj = '';         
// ID объекта, если требуется вывести только конкретный объект/услугу
$folder = 'onriv'; 
// папка где находится система относительно корня
$head = '0';       
// вывод тега <head> '0' - не выводить, рекомендуется, если производится вывод интерфейса на странице сайта, где уже имеется данный тег)
//
include_once($folder.'/index.php');

-----------------------------------------------------

Этот пример реализован в файле "onriv.php". Обращаем внимание, что данный файл должен находиться вне папки "onriv".

-----------------------------------------------------


Так же, разместить интерфейс системы на сайте, можно по средствам iframe.
На пример:

-----------------------------------------------------

<iframe src="http://onriv.com/demo/?obj=object_04_02_2016_11_46_10_67" width="100%" height="800" align="center" frameborder="0"></iframe>

-----------------------------------------------------

Данная конструкция, добавленая в HTML код страницы, выведет на ней интерфейс бронирования указанной услуги (объекта). Меняя значение "src=", можно вывести главное меню всех услуг или только конкретную категорию, указав соответствующий адрес.
